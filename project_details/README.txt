# HMSV2 Project Documentation Index

Welcome to the full technical documentation for the Hostel Management System V2 (HMSV2). 
This folder contains detailed explanations of every major component of the project.

## Document List (Read in this order):

1. **00_PROJECT_OVERVIEW.txt**
   - General summary, tech stack (Laravel 12/PHP 8.2), and core features.

2. **01_CONTROLLERS.txt**
   - Detailed breakdown of all controllers and the business logic they handle.

3. **02_MODELS.txt**
   - Database tables, relationships, and data structure descriptions.

4. **03_ROUTES.txt**
   - URL navigation, middleware (security), and role-based access control.

5. **04_VIEWS.txt**
   - Frontend structure, Blade templates, Alpine.js logic, and UI design patterns.

6. **05_MIGRATIONS.txt**
   - Database version control and history of schema changes.

7. **06_SYSTEM_WORKFLOWS.txt**
   - Step-by-step logic on how features like "Dual Visitor Approval" and "Auto Bed Assignment" actually work.

---

## Why these documents?
These files were created to help developers and stakeholders understand the "Why" and "How" behind the project code. It serves as a manual for maintaining, testing, and expanding the application.
